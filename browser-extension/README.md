# Save to AP Research Vault Browser Extension

This optional unpacked Chrome/Edge extension sends the current page, selected text, links, and image URLs to the AP Research Vault shared inbox.

## Install locally

1. Open `chrome://extensions` or `edge://extensions`.
2. Enable Developer mode.
3. Choose **Load unpacked**.
4. Select this `browser-extension/` folder.

The extension is not published automatically. It opens the same GitHub Pages share-target URL used by the PWA:

`https://anubhaparashar.github.io/personal-knowledge-vault/?share-target=1`
